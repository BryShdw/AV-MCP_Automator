# Modelo de Casos de Uso — AV-MCP Automator
**RUP Fase:** Inicio — IT-1 (Semana 2) | **Va al informe:** ✅ Sí

## Actor principal
**Ingeniero AV** — Usuario que ingresa requerimientos en lenguaje natural y
recibe archivos .cuig listos para abrir y editar en Crestron Construct.

## Casos de uso identificados

| ID | Nombre | Descripción breve |
|---|---|---|
| CU-01 | Generar página | Describe una pantalla; el sistema genera el .cuig |
| CU-02 | Actualizar proyecto | Registra la nueva página en el .cuib maestro |
| CU-03 | Consultar plantilla | Solicita una sala predefinida por caso de uso |
| CU-04 | Validar archivo | Verifica que el .cuig no tiene etiquetas CH5 rotas |
| CU-05 | Conmutar IA | Cambia a Ollama si Gemini no responde |
