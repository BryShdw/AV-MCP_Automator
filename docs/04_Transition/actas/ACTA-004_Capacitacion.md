## Objetivos de la sesión
Transferir el conocimiento operativo del sistema AV-MCP Automator al personal de ingeniería de DACER S.A.C., permitiéndoles instalar, configurar y operar la herramienta para la generación diaria de interfaces visuales de control.

## Temas cubiertos
1. **Instalación:** Despliegue del entorno virtual Python y ejecución de `pip install -r requirements.txt`.
2. **Configuración `.env`:** Asignación segura de la API Key de Gemini en el archivo de variables de entorno.
3. **Selección de proyecto:** Uso del selector de directorios en Streamlit para apuntar a la carpeta del proyecto Crestron Construct activo.
4. **Generación de página:** Redacción de prompts efectivos, selección de temas visuales Pro AV y elección de paneles táctiles.
5. **Apertura en Construct:** Verificación del archivo `.cuig` recién generado en el IDE oficial y confirmación de editabilidad.

## Ejercicios realizados
Durante la sesión, el ingeniero capacitado realizó las siguientes tareas de manera autónoma:
1. Generación de una página de control de volumen para un panel TSW-770 usando la paleta "Corporate Blue".
2. Generación de una página de control de proyector y pantalla motorizada para un panel TST-1080 usando la paleta "Dark Pro".
