

# Z-Order

**Z-order** is the drawing order of overlapping Components on a Project Page. Components that have a higher Z-order are displayed in front of Components that have a lower Z-order. The Z-order determines which Components are visually _on-top_ when Components occupy the same space.

Components at the top of the Z-order are listed at the top of the Layer Manager--similar to layers in illustrator tools:
