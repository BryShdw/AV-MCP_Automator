

# Add a Subcontract

Contracts can aggregate other contracts, which are referred to as subcontracts. Subcontracts are useful when you want to use components from other contracts without having to recreate them from scratch.

To add a subcontract to a new or existing contract:

1. Select the **Contract Details** button from the side navigation menu to open the **Contract Details** screen.
2. Select the **+** (plus) button in the **Subcontract List** section.
3. Select one or more of the recent contracts from the list, or select the ellipses button to open a contract from your workstation.
4. Select **Import**.


Once subcontracts have been imported, the components from each added subcontract can now be added to the primary contract.
