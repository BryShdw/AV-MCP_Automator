

# Modify the AVF-CH5 Project

At a high level, the workflow for modifying the AVF-CH5 project to create a custom .AV Framework UI is as follows:

1. [Download](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/Downloads.htm) and unzip the packaged AVF-CH5 project file to an accessible location on your workstation.
2. Open the project directory in Visual Studio Code.
3. Modify the project theme and styling using the project configuration file.
4. Add or modify project components (such as buttons) and map them to .AV Framework functions using the [AVF-CH5 Contract Reference](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/AV-Framework/API-Reference/API-Reference.htm).
5. Deploy the project to a touch screen within the .AV Framework system.

The AVF-CH5 project has been created using the same Crestron Template Project that is available for standard CH5 projects, which is documented in the [Crestron Template Project](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/Template-Project/Use-Template.htm) topic.

For detailed instructions regarding all development tasks that can be performed to modify the AVF-CH5 project, refer to the [How Do I?](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/Template-Project/How-Do-I.htm) topic.
