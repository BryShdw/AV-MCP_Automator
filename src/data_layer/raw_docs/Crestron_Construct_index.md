

# Overview

Crestron Construct™ Software is a HTML5-based cross-platform developer application. Crestron Construct equips developers with the tools to build integrated, turnkey user interface solutions. Crestron Construct utilizes a WYSIWYG-based, drag-and-drop visual editor.

## System Requirements - Platforms & Operating Systems

- Intel-based Microsoft Windows computers running Windows 10 or later

- Intel-based Apple Macintosh (Mac) computers running OS X 10.16 or later

- Apple Silicon Macintosh (Mac) computers running OS X 10.16 or later
