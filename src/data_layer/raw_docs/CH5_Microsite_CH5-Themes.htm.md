

# CH5 Themes

Crestron CH5 themes allow users to change the look and feel of a project in one location using the latest technologies in web development. Themes can be designed for Crestron touch screen interfaces. Crestron provides sample themes can be modified as needed or used a guide to create new and unique themes. This is accomplished using CSS, which changes the underlying markup structure.

Most theme developers typically add a few changes to a new theme by basing on an existing theme. The CH5 theme architecture is designed in a way where the elements of the base theme are used as a backup if nothing is defined in the new theme.
