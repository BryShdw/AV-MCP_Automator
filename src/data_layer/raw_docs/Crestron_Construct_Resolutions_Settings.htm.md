

# Resolution Settings

Resolutions are initially selected during Solution/Project creation. Supported resolutions can be searched for, filtered, included or excluded.

This allows the user to designate only those target resolutions needed in the active Solution or Project and design resolution-specific Project Pages accordingly.

A Resolution Setting is Project specific and may differ from one Project to another in a Solution with multiple Projects.

## Supported Devices & Resolutions

Supported devices include the following:

- Crestron 60-Series and 70-Series Touchscreens

- Crestron DGE-1000

- Apple iPads and iPhones

- Cisco Navigator

- Google Meet - Lenovo


The specific model and resolution of each supported device is listed in the Edit Device Resolutions window (see below).

## Selecting Resolutions

From the menu bar, select Edit **\>** ResolutionSettings **(Alt+Ctrl+S):**

**![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Resolution%20Settings.jpg)**

**Or** use theProject's right-click menu: **![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Resolution%20Settings%20-%20Right%20Click%20Menu.jpg)**

1. In the Edit Device Resolutions window, un-check the check box in the title bar to deselect all previous selected, if any, resolutions.

By default, the 4K Monitor will remain selected. The 4K Monitor can be unselected after another resolution is selected. At least one resolution is required.



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Edit%20Device%20Resolutions%20-no%20x60.jpg)

2. In the Deselecting device resolutions dialog box, click the OK button:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Deselecting%20device%20resolutions.jpg)

3. Scroll through the listed device resolutions and then select the check boxes for the applicable resolutions that will be used in the Solution or Project **:**![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Selected%20resolutions%20-no%20x60.jpg)

In this example, only the TSW-1070 and the TSW-770 were selected. Click the **X** in upper-right hand to complete the selection and close the Edit Device Resolutions window.



    These two resolutions will be listed in the **Touch** **Screen** drop-down:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Touch%20Screen%20drop-down%20example.jpg)

4. Alternatively, set a filter to display and select the applicable resolutions:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Filter%20by%20width%20-%20no%20x60.jpg)

In this example the Width Filter was selected and the Filter was set to filter for resolutions with a Width of 800px.





![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Resolutions%20found%20-%20no%20x60.jpg)

In this example, the TSW-570 met the Filter criteria and was selected.

5. Click on a column heading's title to sort the resolutions by the corresponding property (ie; Width) in ascending or descending order:



![](https://help.crestron.com/construct/Content/Resources/Images/Sort%20by%20Width.jpg)

In this example, the resolutions will be sorted by Width.

6. Enter a text string in the Search text field to search for matching resolutions:



![](https://help.crestron.com/construct/Content/Resources/Images/Search%20text%20field.jpg)



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Resolution%20Settings/Search%20for%20tsw.jpg)

In this example, the search string: tsw found the corresponding resolutions with tsw in the Device Name.
