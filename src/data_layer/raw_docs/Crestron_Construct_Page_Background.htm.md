

# Theme Page Color (Page Background)

By default, a Project will use the Background Color inherited from the Project Theme.

This Theme Page Color (or Page Background) can be overridden and a new Page Background can be set for all Project Pages.

1. From the Project-level right-click menu, select Project Properties:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Project%20Properties%20drop%20down.jpg)

2. In the Project Properties window, select the Override Theme Color check box:





![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Theme%20Page%20Color%201.41.jpg)




1. In the text field directly below the Override Theme Color check box enter the color name:



      ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Color%20Text%20Field.jpg)

2. Click the right hand drop-down arrow to display the color picker.

3. Click on the color palette or color slider to pick the color; click the APPLY button:



      ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Color%20Picker%20-%20page%20background.jpg)

4. Click the left-hand spin arrows to switch to RGB format:



      ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/RGB.jpg)

      Click the color palette's APPLY button.


Click the Project Properties window's Save Changes button:

![](https://help.crestron.com/construct/Content/Resources/Images/Create%20A%20Solution/Save%20Changes.jpg)

Users can overwrite theTheme Page Color on a page-by-page basis when a new Page is added by selecting the Page Background Color check box and designating a color:

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/New%20Page%20-background%20overwrite.jpg)

1. In the Background Color text field, type the color name:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Page%20Background%20color%20by%20name.jpg)

2. Click on the drop-down arrow and select a Background Color via the color picker:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Color%20Picker.jpg)

3. Click the left-hand Hex Color spinner to enter the RGB settings:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Hex%20Color%20Spinner.jpg)

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Hex%20Color%20Settings.jpg)


Or by editing an existing Page and  selecting the Page Background Color check box and designating a color:

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Edit%20Page%20background%20color.jpg)

Users can apply a background color on a widget-by-widget basis when a new Widget is added or when an existing Widget is edited by selecting the Widget Background Color check box and designating a Background Color:

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Add%20New%20Widget%20Background%20Color.jpg)

1. In the Background Color text field, type the color name:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Add%20New%20Widget%20Background%20Color%20by%20name.jpg)

2. Click on the drop-down arrow and select a Background Color via the color picker:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Widget%20Color%20-%20Color%20Picker.jpg)

3. Click the left-hand Hex Color spinner to enter the RGB settings:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Hex%20Color%20Spinner.jpg)



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Theme%20Page%20Color/Hex%20Color%20Settings.jpg)


## Importing VT Pro Projects - Project-Level Backgrounds

- If a Background Color is set in VT Pro, Construct will retain this Background Color at and after import.

- As expected, a Background Color can be set or overridden after the VT Pro Project is imported.
