

# Date and Time

The <ch5-datetime> component is used to create a controllable widget that displays date and time information that is pushed from a connected control system. The look and feel of the widget is customizable.

For interactive examples using the <ch5-datetime> component and CSS classes, refer to the [Showcase Application](https://sdkcon78221.crestron.com/downloads/ShowcaseApp/ch5-datetime/ch5-datetime-display24-hour-format.html).

## Features

The <ch5-datetime> attributes provide the following features:

- Uses an analog join to programmatically change the time offset in hours property during runtime.
- Allows a combination of date and/or time to be displayed within the CH5 project.
- Provides options for customizing the time display, including offsetting hours, options to display 24-hour format, hiding AM/PM, showing 1 or 2 digits for single-digit hours, and displaying seconds.
- Sets the format for displaying the date and time.
