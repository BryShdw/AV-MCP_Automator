

# Installation

Contract Editor can be installed on a Windows® or macOS® operating system.

The following download options are available for Contract Editor:

- Download directly from the [Crestron website](https://www.crestron.com/support/search-results?c=4&m=10&q=contract)
- Download using the Master Installer tool

## Recommended PC Configurations

The following PC configurations have been tested by Crestron and are recommended for running Contract Editor.

- Recommended Windows PC Configuration:
  - **OS Version**: Windows 10 Home
  - **Processor**: Intel Core® i7-8550U CPU @ 1.80GH 1.80GHz
  - **Memory**: 16 GB
  - **Operating System**: 64-bit, x64-Based Processor
  - **Resolution**: 1920x1080 or better
- Recommended macOS PC Configuration:
  - **OS Version**: macOS Mojave® OS version 10.14.6
  - **Memory**: 16 GB 2400 MHz DDR4
  - **Graphics**: Radeon® Pro 560X 4 GB, Intel® UHD Graphics 630 1536 MB
  - **Resolution**: 1920x1080 or better

## Windows Installation

To install Contract Editor on a Windows operating system.

1. Download and open the CH5 Contract Editor Setup executable. A license agreement message window is displayed.
2. Select **I Agree** to continue.
3. Select the user profile(s) that Contract Editor will be made available to, and then select **Next >**.
4. Choose the destination folder for the Contract Editor installation, and then select **Install**. A status bar is displayed to show the progress of the installation.
5. Once the application is installed, a success message is displayed. Select **Finish** to complete the installation.

## macOS Installation

To install Contract Editor on a macOS operating system.

1. Download and open the CH5 Contract Editor Setup disk image file. A license agreement message window is displayed.
2. Select **Agree** to continue. The disk image file is opened and a setup window is displayed.
3. Within the setup window, drag the **CH5-Contract-Editor** file to the **Applications** directory.
