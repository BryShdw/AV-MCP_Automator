

# Template

The <ch5-template> component is used to create new instances of the user-provided <template> tag and to substitute event and state names where indicated by the integrator.

For interactive examples using the <ch5-template> component and CSS classes, refer to the [Showcase Application](https://sdkcon78221.crestron.com/downloads/ShowcaseApp/ch5-template/validation.html).

## Features

The <ch5-template> attributes provide the following features:

- Sets the ID of the template to be instantiated.
- Sets the replacement value of the user-provided template.

## Design Considerations

To ensure that the template content is displayed as expected, names of elements should not be used in a template substitution.
