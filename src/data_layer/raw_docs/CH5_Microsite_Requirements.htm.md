

# .AV Framework Requirements

The .AV Framework™ software system must be running AVF Pro version 6.9 or higher on one of the following supported .AV Framework controllers:

- DMPS3 Series
- 4-Series® Control Systems
- MPC3-101, MPC3-102, MPC3-201, or MPC3-302
- RMC3
- VC-4 (Crestron Virtual Control)

To enable CH5 user interfaces on the .AV Framework controller, the system configuration must first be changed in the .AV Framework web UI to enable CH5 mode.

To enable CH5 mode in .AV Framework:

1. Log into the .AV Framework web UI as described in [the Operations Guide for your .AV Framework controller](https://www.crestron.com/support/search-results?c=6&m=10&q=.av%20framework).
2. Navigate to **System Setup** > **Touch Screen Custom Graphics**.

Touch Screen Custom Graphics Settings

![](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Resources/Images/AVF-Menu.png)

5. Turn off the **Enable Touch Screen Auto Update** toggle. This function must be turned off to load custom CH5 projects to the touch screen.
6. Use the **.AV Framework User Interface Theme** drop-down menu to select **AVF 2.0 Theme CH5**.
7. Select **Save**.
8. Activate the new configuration.
