

# Create a Project

CH5 projects contain all files and code that are used to create and deploy user interface to touch screens and other outputs. CH5 projects should adhere to all recommended standards as described in [Best Practices](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/UI-Best-Practices.htm)

Crestron provides a standard template project that can be modified to create a custom user interface. For more information on creating, modifying, and deploying a project using the Crestron-provided template, refer to the procedures documented under [Crestron Template Project](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/Template-Project/Use-Template.htm).

NOTE: Crestron strongly recommends using the provided template to create CH5 projects. Crestron support will not be able to diagnose and resolve issues related to custom (non-template) projects.
