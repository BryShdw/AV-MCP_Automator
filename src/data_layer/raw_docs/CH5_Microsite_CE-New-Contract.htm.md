

# Create a New Contract

To create a new contract:

1. From the home screen, select **New Contract** under the **Start** section, or select the **Create a new Contract** button on the top menu. A pop-up dialog box is displayed.

![](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Resources/Images/CE2.png)

3. Enter the following information:
   - Enter a contract name in the **Name** text field.
   - Select the ellipses button to select a location on the workstation where the contract files and output will be saved.
4. Select **Create**. The **Contract Properties** page is displayed.

![](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Resources/Images/CE3.png)

6. Enter the following information as needed:
   - **Name**: Edit the contract name. This field is mandatory.
   - **Description**: Enter a description about the contract contents or purpose.
   - **Author**: Enter the name of the contract author.
   - **Company**: Enter the name of the UI design company or dealer.
   - **Client**: Enter the name of the client requesting the contract.
   - **Version**: Enter a version number of the contract.
   - **File Location**: Modify the location on the workstation where the contract files and output will be saved
7. Select the build options for the contract ( **SIMPL** and/or **SIMPL# PRO**). When the contract specification is built, it will build files that are compatible with the selected programming environments.
8. Add any subcontracts from within the **Subcontracts** section. For more information, refer to [Add a Subcontract](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/CE-Subcontract.htm).

Once a contract is created, you can add programming components to the contract.
