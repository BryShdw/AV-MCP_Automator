

# Touch Screen Emulation

Developing iterative touch screen projects can be time consuming if updates are reviewed on the touch screen. However, the Dev Tools feature within the Chrome® web browser allows developers to emulate mobile devices, which can be leveraged to provide accurate review of your project layout and rendering on your workstation.

For information on how to configure Chrome, refer to [https://developers.google.com/web/tools/chrome-devtools/device-mode/#custom](https://developers.google.com/web/tools/chrome-devtools/device-mode/#custom).

NOTE: To ensure that the emulation is functioning correctly, refresh the Chrome web browser after entering emulation mode.

## TSW-60 Series Emulations

Use the settings provided below to emulate screen rendering of a TSW-60 series device.

| Device Name | Width | Height | Device Pixel Ratio | Device Type |
| --- | --- | --- | --- | --- |
| TSW-1060(-NC) | 1280 pixels | 800 pixels | 1 | Mobile |
| TSW-760(-NC) | 1261 pixels | 739 pixels | 0.8125 | Mobile |
| TSW-560(-NC) | 640 pixels | 363 pixels | 1.5 | Mobile |
| TSW-560P | 363 pixels | 640 pixels | 1.5 | Mobile |
