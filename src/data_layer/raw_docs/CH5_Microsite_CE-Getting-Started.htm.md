

# Getting Started

Once Contract Editor has been installed to a workstation, open the application. The home screen is displayed.

![](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Resources/Images/CE1.png)

The home screen contains three primary sections:

- **Start**: The **Start** section provides tools [to create a new contract](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/CE-New-Contract.htm), t [o open an existing contract](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/CE-Open-Contract.htm), and [to import a contract](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/CE-Import-Export.htm).
- **Recent Contracts**: The **Recent Contracts** section displays a list of recent contract projects for quick access.
- **Short Cuts**: The **Short Cuts** section displays a list of short cut keys that can be used to perform various application function.

Help buttons are provided in various areas of the application. Click the button to learn more about the associated feature.

## Navigation

Contract Editor provides a side menu that contains the following selections. The menu is available on any page within the application.

- **Contract Properties**: Opens a screen where contract information can be edited.
- **Component Details**: Opens a screen where components can be added and edited.
- **Contract**: Opens a screen where the specification can be edited.

Additionally, a toolbar is provided at the top of the application that contains the following selections. This toolbar is available on any page within the application.

- **Home**: Returns to the home screen
- **Create a new Contract**: Creates a new contract
- **Open an existing Contract**: Opens an existing contract
- **Save Contract**: Saves the contract that is being modified
- **Save As**: Saves the contract that is being modified with an option to rename the saved file.
- **Recent Contracts**: Displays a drop-down list to select recent contract projects
- **Import Archived Contract**: Imports an archived contract to a specified destination
- **Archived Contract**: Exports a contract
- **Undo**: Undoes the last performed action
- **Redo**: Redoes the last undone action
- **Build Contract**: Builds the contract specification for use in the chosen programming environment(s)
- **Preferences**: Provides selections for disabling tool tips and choosing a language for the application.
- **About**: Displays information about the application, including the version.

If the application screen is minimized, some of the selections above are placed into a drop‑down menu on the top left of the application. Click the hamburger menu icon to access these selections.
