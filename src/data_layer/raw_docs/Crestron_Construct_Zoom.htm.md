

# Zoom

During page design, the user can zoom in and zoom out on the active Page.

## Zooming In & Zooming Out

1. Select the applicable Page.

2. On the Toolbar, click on the drop-down arrow to the right of the magnifying glass icon:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Zoom/zoom%20smart%20incon.jpg)

3. Select the Zoom (magnification) level from the drop list:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Zoom/Zoom%20drop%20down%20list.jpg)

Pre-defined Zoom levels include Fit All (default) and range from 25% to 750%.



Click the magnifying glass smart icon to reset the Zoom level to 100%:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Zoom/100%20percent.jpg)



In the Zoom smart icon text field, press Backspace and enter a custom Zoom level, if applicable:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Zoom/Custom%20Zoom.jpg)

4. Two keyboard+mouse Zoom combinations are also available:


   - On Microsoft-based computers, press Ctrl+Mouse Wheel.
     - Roll the Mouse Wheel forward (away from the user) to increase the Zoom level incrementally.

     - Roll the Mouse Wheel backward (toward the user ) to decrease the Zoom level incrementally.

   - On Apple-based computers, press Command+Mouse Wheel.
     - Roll the Mouse Wheel forward (away from the user) to increase the Zoom incrementally.

     - Roll the Mouse Wheel backward (toward the user ) to decrease the Zoom incrementally.

### Fit All

- When Fit All is applied to a Page, the view zooms to fill the available space based on the device dimensions.

- However, when Fit All is applied to a Widget, Fit All adjusts the Zoom level to fill the Widget container.

- Fit All Shortcut Keys:
  - On Microsoft computers, press Ctrl+Alt+F.

  - On Apple computers, press Command+Option+F.

## Zoom: Switching Between Resolutions

Different Zoom levels can be set for each Resolution in the Project. Each Resolution's Zoom level is maintained when switching between Project Resolutions.

For example, on a 4K Monitor the Zoom level was set to 125%; on a TST-1080, it was set to 200%.

If user switches to the TST-1080, its Zoom level will be 200% as expected. When the user switches back to the 4K Monitor, its Zoom level will be 125% as expected.

The Zoom levels are maintained between Resolutions.

## Zoom: Switching Between Editor Page Tabs

Different Zoom levels can be set for each Project Page. Each Page's Zoom level is maintained when switching between Project Pages.

For example, on the Main Page the Zoom level was set to 150%; on the Lab Access Page, it was set to 175%.

If the user switches to the Lab Access Page via selecting its tab on the Page Editor, its Zoom level will be 175% as expected. When the user switches back to the Main Page via selecting its tab on the Page Editor, its Zoom level will be 150% as expected.

The Zoom levels are maintained between Pages when switching between Pages via their corresponding Editor Page Tab.

The Zoom levels are also maintained for between Widgets when switching between Widgets via their corresponding Editor Page Tab.

The only distinction pertains to Fit All--while it adjusts to the device dimensions for the Page, it adapts based on the Widget container for the Widget.
