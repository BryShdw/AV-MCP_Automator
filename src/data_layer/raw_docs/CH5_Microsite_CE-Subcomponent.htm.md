

# Add a Subcomponent

Components can aggregate other components, which are referred to as subcomponents (or reference components). Subcontracts are useful when you want to add existing components to new components in the programming output.

To add a subcomponent to a component:

1. In the **Reference Components** section of the component, use the drop-down menu under **Component Name** to select the desired subcomponent.
2. Select the text field under **Instance Name** to modify the name of the instance if necessary.
3. Select the text field under **Number of Instances** to set the number of instances of the subcomponent within its parent component.
