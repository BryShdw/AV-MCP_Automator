

# Touch Screen Deployment

The following topics describe considerations and features related to deploying your CH5 project on a touch screen.

## Supported Control Systems

Refer to the following table for a list of control systems that are supported for deploying CH5 project to touch screens.

| Control System | Touch Screen Support? |
| --- | --- |
| 4-Series® Control Systems | Yes |
| 3-Series® Control Systems | Yes |
| Crestron Virtual Control (VC-4 Series) | Yes |
| DMPS3 Series | Yes |
