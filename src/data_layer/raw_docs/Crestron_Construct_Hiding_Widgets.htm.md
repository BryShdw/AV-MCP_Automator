

# Hiding Widgets

Widgets on a Project Page can be hidden during both Page design and runtime.

## Page Design

Hide Widgets, as needed, during Page design to work more efficiently while designing the Project Pages. This can prevent Widgets from overlapping.

If a Widget does not have a value set for its Receive Digital Feedback – Visibility Property, then hiding the Widget during Page design will also hide the Widget at runtime. This can be useful when working with multiple resolutions when a Widget needs to be visible in one resolution and hidden in another.

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Hiding%20Widgets/V2/Receive%20Feedback%20Visibility%20Property.jpg)

In this example, the Receive Digital Feedback - Visibility Property has not been assigned a value. Therefore, this will hide the Widget during both Page design and runtime.

### Hiding All Widgets

1. Click on the Page Body (canvas) to give it the active focus. The purple frame indicates that the Body has the active focus.



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Hiding%20Widgets/Active%20Focus.jpg)

2. Press Alt+H on a Window-based computer to hide **all** the Widgets on the current Page:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Hiding%20Widgets/V2/All%20Widgets%20hidden.jpg)

In this example, all the Widgets on the current Page were hidden by pressing Alt+H.

3. Press Option+H on an Apple-based computer to hide **all** the Widgets on the current Page.


### Hiding & Showing Individual Widgets

On a Project Page, individual Widgets can be hidden and shown as applicable.

1. In the Layer Manager, click the Hide/Show smart icon (eye ball) to the left of the Widget to toggle its visibility on and off:



![](https://help.crestron.com/construct/Content/Resources/Images/Eyeball.jpg)
