

# Click Assign

Click Assign allows the user to set input and output joins automatically for the following Components:

- Button

- Color Chip

- Formatted Text

- Image

- Segmented Gauge

- Slider

- Textinput

- Toggle

- Wifi Signal Level Gauge


## Setting Joins Automatically

1. To set input or output joins automatically, click on the Digital Assign,Analog Assign or Serial Assignsmarticons, as applicable on the UI Editor Toolbar:



![](https://help.crestron.com/construct/Content/Resources/Images/Click%20Assign.jpg)

This puts the application workspace into Click Assign mode and disables all the Managers until the Esc key is pressed or another Project Page is selected.





![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Click%20Assign/Click%20Assign%20Workspace.jpg)

In this example, the application workspace is in Click Assign mode. The Layer Manager, PropertiesGrid  and Block Manager are disabled and grayed-out. To exit Click Assign mode, press the Esc key or select another Project Page **.**


1. By default, joins will be set starting at the first unused join. Otherwise, select the applicable join assignment radio button and click the SET button (see below):



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Click%20Assign/Set%20Digital%20Joins.jpg)

Digital Assign, Analog Assign and Serial Assign have identical join assignment radio buttons/options.


   - **Start at the First Unused Join**: starts automatic join assignment at the first unused join by the join type: Digital, Analog or **Serial**.

   - **Start at the First Unused Join Greater than the Largest Used Join**: starts automatic join assignment at the first unused join greater than the largest used join number by the join type.
   - **Start at the Selected Join:** starts automatic join assignment at the join number designated in the Starting Join text field:



     ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Click%20Assign/Starting%20Join.jpg)

   - **Always Use the Selected Join:** always use the selected join number for every join assignment.
2. Click on each Component on the Project Page to set the joins based on the join assignment option selected above.

3. If the Component already has been assigned a join, then in the Object Already Joined window is displayed:



![](https://help.crestron.com/construct/Content/Resources/Images/Object%20Already%20Joined.jpg)

4. Click the REPLACE button to replace the existing join with an automatically assigned join -or- click the CANCEL to retain the existing join.

5. To exit ClickAssign press the Esc key or select another ProjectPage.
