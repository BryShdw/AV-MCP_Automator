

# Overview

The [Crestron ONE app](https://www.crestron.com/Products/Control-Hardware-Software/Software/Apps/CRESTRON-ONE) for Apple® iPhone® and iPad® devices provides various user control solutions for a mobile device. When set to HTML5 UI mode, the Crestron ONE app enables custom Crestron HTML5 User Interface projects to be loaded into the application, creating a personalized user control experience for the home or office.

Programming for the Crestron ONE app uses the same Crestron HTML5 User Interface scripts and components needed for touch screen and HTML Web XPanel deployments.

The topics that follow describe how to program, configure, and deploy the Crestron ONE app functionality using a Crestron control system.

## Supported Mobile Platforms

Refer to [Knowledge Article 5655](https://community.crestron.com/s/article/id-5655 "https://community.crestron.com/s/article/id-5655") for a list of mobile platforms that are supported by the Crestron ONE app.

## Supported Control Systems

| Control System | Crestron ONE App Support? |
| --- | --- |
| 4-Series® Control Systems | Yes |
| 3-Series® Control Systems | Yes |
| Crestron Virtual Control (VC-4 Series) | Yes |
| DMPS3 Series | Yes |

NOTE: The MC3 and TPCS-4SM 3-Series control systems do not support the Crestron ONE App functionality.
