

# XPANEL, CRESTRON ONE & Web Browsers

## XPANEL

XPANEL transforms any computer into a virtual Crestron Touchscreen. It can be installed and run as a desktop application on a laptop or desktop computer running Windows® or macOS®.

XPANEL is well-suited for testing and fine tuning Crestron Construct UIs when used with Chrome, Edge, Safari, Firefox or most modern browsers.

Note: Construct UIs can only run as Xpanels using a browser.

## CRESTRON ONE

CRESTRON ONE is a mobile app for Apple® iPhone® and iPad® and Android-based devices that enables Crestron Construct UIs to be deployed to the mobile devices.

## Web Browsers

Browsers equipped with developer tools are excellent tools for testing because they allow the user to switch between different panel types/resolutions quickly without the physical hardware.

In most browsers, press F12 to enter developer mode. In developer mode, press Shift+Ctrl+M to display the Device Toolbar Dimensions drop-down. This allows the user to choose and simulate various Viewport sizes from a pick list. In addition the user, typically, can edit the pick list and add custom device viewport sizes.

Alternatively, right-click anywhere the Project Page and select Inspect from the menu to open the developer tools.
