

# Copy & Paste

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/copy%20and%20paste%20icons.jpg)

In a Solution, Project Pages, Widgets and Assets can be copied from one Project (Source) to another Project (Target).

## Copying & Pasting Project Pages

1. In the Solution Explorer tree, select and expand the Source Project to display its Pages, Widgets and Assets.

2. Single-click on the applicable Project Page to select it:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Source%20Page%20Select.jpg)

In this example, the Project Page: Main Menu has been selected as indicated by the blue-gray horizontal bar. This Project Page includes the Widget: Environment and the Asset: Files. These will be copied to the Target Project when the Project Page is pasted to it--as well as any Components, if present on the Project Page.

3. Click on the Copy smart icon below the Menu bar or press Ctlr+C to copy the entire Project Page:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Copy%20Icon.jpg)

4. In the Solution Explorer Tree, open and expand the Target Project; single-click on the Pages folder:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Select%20Target.jpg)

**Note:** the blue-gray horizontal bar indicates which folder in the Solution Explorer Tree has the current active focus.

5. Click on the Paste smart icon below the Menu bar or press Ctlr+V to paste the entire Project Page to the Target Project:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Paste%20Icon.jpg)

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/After%20Paste.jpg)

In this example the Project Page has been copied and pasted and includes the Components, Widgets and Assets from the Source Project Page:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Pasted%20Page.jpg)

On the Target Project, the pasted Project Page includes the following Components: Formatted Text, Buttons, a Slider, a Toggle, and Textinput. In addition, the Image Component uses the Files Asset and the custom Environment Widget has been pasted.


## Copying & Pasting Widgets and Assets

Widgets and Assets can be separately copied and pasted from the Source Project to the Target Project without coping and pasting a Project Page. For example, a Widget or Asset might to copied and pasted to a Target Project and then used on a Project Page unique to that Target Project.

01. In the Solution Explorer tree, select and expand the Source Project to display its Widgets and Assets.

02. Single-click on the applicable Widget or Asset to select it:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Widget%20Selection.jpg)

    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Asset%20Selection.jpg)

03. Click on the Copy smart icon below the Menu bar or press Ctlr+C to copy the Widget or Asset:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Copy%20Icon.jpg)

04. In the Solution Explorer tree, select and expand the Target Project; single-click on the Widgets folder or the Assets folder as applicable:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Widgets%20Folder.jpg)

    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Assets%20Folder.jpg)

05. Click on the Paste smart icon below the Menu bar or press Ctlr+V to paste the Widget or Asset to the Target Project:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Paste%20Icon.jpg)

    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Widgets%20and%20Assets%20Pasted.jpg)

06. In the Solution Explorer tree, press Ctrl+Single-click or Shift+Single-click to select multiple Project Pages, Widgets or Assets in the Source Project:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/MultiSelect.jpg)

    In this example, a Widget and two Assets have been multi-selected in the Source Project via Ctrl+Single-click.

07. Click on the Copy smart icon below the Menu bar or press Ctlr+C to copy the multi-selected items:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Copy%20Icon.jpg)

08. In the Solution Explorer tree, select and expand the Target Project:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/MultiSelect%20Paste%20to%20Target.jpg)

09. Click on the Paste smart icon below the Menu bar or press Ctlr+V to paste the multi-selected items to the Target Project:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Paste%20Icon.jpg)

    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Pasted%20to%20Target%20Example.jpg)

10. If an identical Project Page, Widget or Asset already exists in the Target Project, then the Paste Items into Project dialog box is displayed:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Paste%20to%20New%20Project%20dialog%20box.jpg)

11. Under Select All click the following buttons to apply the following Paste Conflict resolution to all the listed items:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/Select%20All.jpg)


    - **Replace:** Replace the existing items with the Copied items and thereby overwrite the existing items in the Target Project.

    - **Use Existing:** Use the existing items in the Target Project as is. This cancels the Paste operation.

    - **Create New:** Rename the Copied items; thereby, effectively making them new, unique items to the Target Project.  Create New automatically appends a "-#" to the Type Name. For example, Hour Glass would automatically be renamed as Hour Glass-1 . A new name can be typed ; for example: Timer:



      ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Select%20All%20Create%20New%20Append.jpg)

      In this example, "-1" was appended to the Asset and Widget names.





      ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Select%20All%20Create%20New%20Rename.jpg)

      In this example, the Asset: Hour Glass will be renamed to Timer.
12. Click the Paste Items button to Paste the Copied items to the Target Project now that the Paste Conflicts are resolved:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/Paste%20Items.jpg)

13. Paste Conflicts can also be resolved on an item-by-item basis:



    ![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Individual%20Paste%20Conflicts.jpg)

    **Note:** The Paste Items into Project dialog box tracks items present on the Clipboard slated to be pasted. If the Clipboard changes before conflicts in the Paste Items into Project dialog box are resolved, then the Paste operation will be canceled and the dialog box will be closed automatically.


## Copying & Pasting Components

Individual or groups of Components can be copied and pasted from a Source Project Page to a Target Project Page without copying and pasting the Source Project Page. However, the process is slightly different and centers on a Page Canvas-level active focus.

### Page Canvas Active Focus

The Source Project Page needs to be opened and the applicable Components selected and then copied. Similarly, the Target Project Page needs to be opened first before pasting the Components. The Components will retain all their original Property settings when pasted to the Target Project Page--including Size, Position and Color.

1. Open the Source Project Page.

2. Select the applicable Components:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Component%20Copy%20Source.jpg)

3. Click on the Copy smart icon below the Menu bar or press Ctlr+C to copy the Components:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Copy%20Icon.jpg)

4. Open the Target Project Page:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Target%20Page%20Component%20Not%20Pasted%20Yet.jpg)

5. Click on the Paste smart icon below the Menu bar or press Ctlr+V to paste the Components to the Target Project Page:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Paste%20Icon.jpg)

![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Copy%20&%20Paste/V2/Components%20Pasted%20to%20Target%20Page.jpg)

All the copied and pasted Components have been left intact--noticeably their Position, Size and Color.
