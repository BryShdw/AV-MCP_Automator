

# Contract Editor

The Crestron® Contract Editor tool is a HTML5 tool set used to create and program touch screen user interfaces. Contract Editor is designed to supersede Smart Graphics® technology and VT Pro-e® software.

Contract Editor was created to satisfy the following objectives:

- To provide touch screen UI developers with a standard technology for development
- To allow UI developers to create designs without knowing specific programming details (such as join numbers)
- To provide a mechanism that enables UI developers and control system programmers to communicate the interaction of their respective components
