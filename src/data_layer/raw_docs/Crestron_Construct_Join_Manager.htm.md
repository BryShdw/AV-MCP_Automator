

# Join Manager

The Join Manager displays the Digital, Analog and **Serials** join numbers that are assigned across all Pages, WidgetComponents and Hard Buttons in the Solution--as well as available join numbers. In the JoinManager, these Pages, Widgets and Hard Buttons are listed in the Pages, Widgets, and Hard Buttons window by name.

## Viewing Assigned Joins

1. From the menu bar, select Tools **\>** Join Manager:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Join%20Manager/Join%20Manager%20menu%20bar.jpg)

2. By default, the **Digital** join numbers are initially displayed. Click on the Analog or **Serial** buttons to display the corresponding join numbers:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Join%20Manager/Join%20Manager.jpg)



In this example, the Digital join numbers are displayed. The Page **:** Main Menu is listed because it has **Digital** join numbers assigned. The custom Widget **:** Environment is listed because it has a **Digital** join number assigned to it

The Digital join numbers 1 and 2 are assigned and highlighted in blue. The light blue join numbers are available and can be assigned during Page Design and Component configuration.

3. Click on a join number to list the Page or Widget where it has be assigned:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Join%20Manager/Join%20Manager%20Show%20Page.jpg)

In this example, the user clicked on the **Digital** join number **2**. The join number is framed in green. The Pages, Widgets, and Hard Buttons window title appended a **\- 2** to its title to denoted that this join number has the active focus. The Page: Access Panel is listed; this indicates that the **Digital** join number **2** is assigned on this Page.

4. In the Pages **,** Widgets, and Hard Buttons window, double-click on the Page, Widget or Hard Button to display it:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Join%20Manager/Display%20Page%20with%20Join.jpg)

In this example, the Page: Access Panel is displayed and the Digital join number **2** has been to the Image Component's Visibility input join.
