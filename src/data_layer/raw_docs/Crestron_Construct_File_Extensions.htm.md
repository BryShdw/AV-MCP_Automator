

# Files, File Extensions & Default Paths

|     |     |     |     |     |
| --- | --- | --- | --- | --- |
| File | File Extension | Windows Default Path | MAC Default Path | Comments |
| Crestron Construct Solution | .csln | Documents/Crestron/Construct/Solutions | Documents\\Crestron\\Construct\\Solutions | Main Solutions folder for both Windows and Mac |
| Contract File | .chd | Output/Contract/Programming/SIMPL | Output\\Contract\\Programming\\SIMPL | N/A |
| CH5 Project File | .cuip | Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\] | Crestron Construct\\Solutions\\\[Solution Name\]\\\[Project Name\] | N/A |
| Crestron Construct Archive | .cca | Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\] | Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\] | Use the archiver file format (.cca) to load a project to a processor web server. |
| Custom Fonts | \*.ttf, \*.eot, \*.woff, \*.woff2, \*.svg | Documents/Crestron/Crestron Construct/Web Fonts | Documents\\Crestron\\Crestron Construct\\Web Fonts | Construct must be restarted after adding custom fonts. |
| Theme Files | \*.zip | Documents/Crestron/Crestron Construct/Themes | Documents\\Crestron\\Crestron Construct\\Themes | Theme files are compressed in .zip format.<br>Construct must be restarted after adding custom themes. |
| Asset | \*.cuia | Documents/Crestron/Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\]/assets | Documents\\Crestron\\Crestron Construct\\Solutions\\\[Solution Name\]\\\[Project Name\]\\assets | N/A |
| Page | \*.cuig | Documents/Crestron/Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\] | Documents\\Crestron\\Crestron Construct\\Solutions\\\[Solution Name\]\\\[Project Name\] | N/A |
| Widget | \*.cuiw | Documents/Crestron/Crestron Construct/Solutions/\[Solution Name\]/\[Project Name\] | Documents\\Crestron\\Crestron Construct\\Solutions\\\[Solution Name\]\\\[Project Name\] | N/A |
