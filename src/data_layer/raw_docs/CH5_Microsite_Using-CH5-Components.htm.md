

# CH5 Components and Themes

CH5 components are custom HTML elements provided by Crestron. The primary purpose of CH5 components is to provide messages to and from the control system. In CH5, these messages are called events and states, and they are sent via the same Joins mechanism that is currently used in Smart Graphics® technology based user interfaces.

CH5 themes are CSS styles that are used to control the look and feel of CH5 components within a project.
