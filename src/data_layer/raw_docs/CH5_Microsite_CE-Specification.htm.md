

# Build a Contract

Once components have been added to a contract, the contract can be built and outputted to the chosen programming environment.

To build a specification:

1. Select the **Contract** button from the side navigation menu to open the **Contract** screen. All components that have been added to the contract are displayed.
2. Enter a name for the instance in the **Instance Name** text field.
3. Enter the number of instances in the **Instances** text field. The instance will be repeated _x_ number of times in the programming output, where _x_ is the number entered.
4. Repeat steps 2–4 for each additional component that will be added to the contract.
5. Select the **Build Contract** button from the top menu. If the build is successful, a success message is displayed.

To remove a component from a contract, select the trash can icon in the component row.

Depending on the selected programming outputs (specified when creating or editing the contract), output folders with the following files are created under the \[SaveFile\] > **output** folder.

- **SIMPL**: .chd file
- **SIMPL SHARP PRO**: .g.cs files
- **CH5**: .json file
