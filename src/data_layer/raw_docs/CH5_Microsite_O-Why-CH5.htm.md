

# Why Should I Use Crestron HTML5 User Interface?

Crestron® HTML5 User Interface offers the following benefits to dealers, integrators, and web developers:

- Offers an alternative to VT Pro-e® and Smart Graphics® programs that is built on existing web technologies.
- Projects are future proof as the web evolves; Crestron HTML5 User Interface uses open web standards.
- Projects can be extended using open source and third-party solutions.
- Projects can be built once and deployed to many display types and aspect ratios (in landscape and portrait modes)
- Crestron HTML5 User Interface is supported on Crestron TSW‑60, TSW‑70 series, and TST‑1080 touch screens, as well as the DGE‑1000 using a connected touch screen display

## Usability

- Build rich and dynamic user interfaces.
- Present a modern design appearance to meet customer requirements.
- Present a common look and feel across device types.
- Projects can be easily themed and customized.
- Add localization quickly.

## Productivity

- A single, developed project works across many screen sizes and device types (panels and mobile).
- Front-end and back-end design can occur independently:
  - Programmers can build efficient and reusable back ends.
  - Developers can build flexible and reusable front ends.
- Code can be reused and is scalable

## Technology

- Crestron HTML5 User Interface is built on open standards, allowing multiple ways to deliver.
- Leverage open source and third-party solutions and components.
- Crestron HTML5 User Interface uses a widely-used and familiar web development tool chain.
- Crestron HTML5 User Interface is extensible with web technologies.
- Crestron HTML5 User Interface provides an SDK and API to interface with Crestron solutions for out-of-the-box development.
