

# Create UI Elements

Crestron HTML5 User Interface allows developers to create components and widgets that represent graphical elements in a user interface, such as buttons, keypads, and text. CH5 offers many of the same UI elements that could be created in Smart Graphics, with new elements scheduled for future releases.

NOTE: For more information on CH5 components, refer to [Components](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/UI-Components.htm).
