

# Video Considerations

Whenever the <ch5-video> component is included in a project, do not use any HTML or CSS background functions. These background functions will cover up the video that is playing behind.

To add a background, use the <ch5-background> component at the page or project level.

* * *

The size of the CH5 video is based on your device view port size. Whenever stretch is enabled, the display size of the video is calculated based on the width and height of the parent element.
