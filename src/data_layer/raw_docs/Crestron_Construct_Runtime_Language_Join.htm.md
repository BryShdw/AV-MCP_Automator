

# Runtime Language Join

Available in CH5 Version 2.13 or later, the Runtime Language Join is used to set a Serial join signal that dynamically changes the language file used with the CH5 project at runtime. The Runtime Language Join is programmed to pass through the language file name.  The UI text will be changed as per the language selected.

## Setting the Runtime Language Join

1. From the Project's right-click menu, select Project Properties:



![](https://help.crestron.com/construct/Content/Resources/Images/Runtime%20Language%20Join/Project%20Properties.jpg)

2. In Project Properties window, enter a serial join number in the Runtime Language Join text field:



![](https://help.crestron.com/construct/Content/Resources/Images/Runtime%20Language%20Join/Runtime%20Language%20Join.jpg)

Alternatively, click the **#** smart icon to display the Serial Join Table and select an available serial join number:

![](https://help.crestron.com/construct/Content/Resources/Images/Runtime%20Language%20Join/Smart%20icon.jpg)

![](https://help.crestron.com/construct/Content/Resources/Images/Runtime%20Language%20Join/Join%20Table.jpg)

3. Click Save Changes:



![](https://help.crestron.com/construct/Content/Resources/Images/Runtime%20Language%20Join/Project%20Properties%20-%20%20Save%20Changes.jpg)
