

# Showcase Application

The [Showcase Application](https://sdkcon78221.crestron.com/downloads/ShowcaseApp/index.html) provides a sandbox environment for the CH5 component library that provides interactive examples showing how components are used within CH5 projects.

NOTE: As of September 2023, the standalone component reference documentation has been merged with the Showcase Application. This documentation provides a reference for components and custom attributes, which are used to communicate front‑end UI design with back‑end programming on Crestron devices.

![Showcase Application](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Resources/Images/Showcase-App-Home.png)

Use the left-hand menu in the Showcase Application to navigate to various examples that showcase the behavior and functionality of each CH5 component. Refer to the read me instructions in the application for more information.

NOTE: Some example pages start with an emulator section, which contains a JSON that defines the emulator scenario/logic. The ins and outs are displayed upon expanding the **Emulator Scenario** tab.

As of June 2025, a Theme Builder extension has been added that allows developers to build custom themes using existing CH5 theme files. For more information, refer to [Theme Builder](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/Theme-Builder.htm).
