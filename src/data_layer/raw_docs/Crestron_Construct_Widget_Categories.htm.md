

# Creating Widget Categories

Widgets can be assigned, organized and displayed in Widget Categories.

Best practice dictates that they are organized by their function. For example: Lights.

## Assigning Widget Categories

1. From the Project Tree's Widget folder, select the Widget:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Widget%20Categories/Select%20Widget.jpg)

2. In the Properties Grid, type the applicable Widget Category name in the Category property:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Widget%20Categories/Widget%20Category.jpg)

This will create the Widget Category.



If the Widget Category already exists, then select it from the drop-down:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Widget%20Categories/Widget%20Category%20Drop%20Down.jpg)



Widget Categories are displayed in the Block Manager:



![](https://help.crestron.com/construct/Content/Resources/Images/Project%20&%20Pages/Widget%20Categories/Widgets%20Block%20Manager.jpg)
