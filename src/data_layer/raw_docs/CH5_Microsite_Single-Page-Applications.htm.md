

# Single-Page Applications

Single Page Applications (SPAs) are fast and responsive because they do not update the entire page but only the required content. Therefore, SPAs significantly improve the speed of the application. Most resources (HTML, CSS, and Scripts) are only loaded once throughout the lifespan of an application. Only required data is transmitted back and forth between the control system and the application loaded on the device.
