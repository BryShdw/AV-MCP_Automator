## Solution

A Solution is a collection of one or more UI projects. For example, a HomeSolution might include separate UI projects for the touch screens or devices used to control the devices in each room or living space. TheLiving Room UIproject could control the Smart TV, sound system, lighting and shades in the living room. A Game Room UI project could control the lighting and sound system in the game room.

![](https://help.crestron.com/construct/Content/Resources/Images/Create%20A%20Solution/Solutions%20and%20Projects.jpg)

## Project

A Project is a single UI design for a touch screen or device.
