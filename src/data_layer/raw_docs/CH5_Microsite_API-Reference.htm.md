

# AVF-CH5 Contract Reference

The topics within this section provide an overview of the UI technical architecture for .AV Framework™ software (AVF) when using Crestron HTML5 User Interface (CH5), including a high-level description of the of the AVF-CH5 contract architecture, and the functionality supported by the system. This information is designed to guide development of the programming content in terms of messages and services.

The contract described in this section has been incorporated into AVF using the CH5 Contract Editor. The output of the editor can be used to produce a CH5 user interface on a touch screen. Refer to [Contract Editor](https://sdkcon78221.crestron.com/sdk/Crestron_HTML5UI/Content/Topics/CE-Overview.htm) for more information on using the software.

- The AVF-CH5 contract provides the API for communication between the AVF control program and a touch screen. This contract was designed for use with CH5 components and is provided as output from the Contract Editor.
- The AVF-CH5 contract has been designed to provide a programming interface for touch screen programming. The AVF contract was created using the Contract Editor. The SIMPL# output files from the Contract Editor have been integrated with AVF, and a description of those integrations are described in the following topics.
